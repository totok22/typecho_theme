# 已完成任务

1. ✅ 目录和搜索等操作添加快捷键
   - `/` 聚焦搜索框
   - `T` 打开/关闭目录
   - `M` 切换主题模式
   - `G` 返回顶部
   - `Esc` 关闭目录面板

2. ✅ 文章正文标题样式优化
   - 一级标题：左侧蓝色竖线装饰 + 底部边框

   - 三级标题：箭头装饰前缀（已移除箭头，改为简洁样式）

   - 所有标题靠左对齐，无缩进

3. ✅ 文章正文标题添加折叠功能
   - 每个标题前添加竖线折叠按钮
   - 不同级别标题竖线颜色不同（H1蓝、H2绿、H3橙、H4紫、H5红、H6灰）

   - 点击可折叠/展开该标题下的内容

4. ✅ Bug修复 - 目录点击最后一个标题时高亮显示错误
   - 重写了高亮逻辑，使用更准确的匹配算法
   - 点击目录项时立即更新高亮状态
   - 添加滚动锁机制，防止滚动事件覆盖点击高亮

5. ✅ 目录添加常驻按钮以及一键全部展开和折叠按钮
   - 常驻按钮：点击后目录面板保持打开状态
   - 展开/折叠按钮：合并为一个按钮，点击切换全部展开/折叠状态

6. ✅ 快捷键提示显示
   - 搜索框内显示 `/` 快捷键提示

   - 所有按钮添加工具提示，显示功能和对应快捷键

8. ✅ 阅读进度条功能
   - 页面顶部显示阅读进度条
